# Venue Intelligence System (VIS)

Real-time venue enrichment tool for live event operators.
Built for the USC Marshall MBA Tixr Case Competition — Team Fanatic Four.

Live demo: [link if available]

## What it does

VIS pulls real-time venue data using the Ticketmaster Discovery API and enriches
it with contextual signals to help event operators make faster, smarter decisions
about venue selection, capacity planning, and market fit.

Designed with Singapore as the beachhead market for international expansion.

## The problem it solves

Live event operators lack a unified, real-time view of venue performance data
across markets. VIS surfaces enriched venue intelligence in one place, reducing
the manual research burden for operators entering new markets.

## Stack

- Ticketmaster Discovery API
- Supabase
- React + TypeScript
- Lovable
